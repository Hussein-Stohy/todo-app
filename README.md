# Todo App

A simple and elegant Todo application built with Angular 18.

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete
- 📱 Responsive design

## 🛠️ Technologies Used

- **Angular 18** - Frontend framework
- **TypeScript** - Programming language
- **CSS3** - Styling
- **HTML5** - Markup

## 📋 Prerequisites

Before running this project, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- [Angular CLI](https://angular.io/cli) (install globally with `npm install -g @angular/cli`)

## 🔧 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hussein-Stohy/todo-app.git
   cd todo-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

## 🏃‍♂️ Running the Application

### Development Server
```bash
npm start
```
Navigate to `http://localhost:4200/` in your browser. The application will automatically reload when you make changes to the source files.

### Build for Production
```bash
npm run build
```
The build artifacts will be stored in the `dist/` directory.

### Running Tests
```bash
npm test
```

### Build and Watch (Development)
```bash
npm run watch
```
